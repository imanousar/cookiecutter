# cookiecutter

A minimalist cookiecutter for data science projects.
<br>

Create project:

```cmd
cookiecutter gh:imanousar/cookiecutter
```

Create venv(Win):

```
python -m venv venv
source venv/Scripts/activate
```

Install requirements:

```
pip install -r requirements
```

Fix notebooks:

```
./nbqa.sh
```
